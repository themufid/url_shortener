# URL Shortener App

A simple URL shortener application built with Python that runs in the terminal.

## Features
- URL shortening
- Custom alias
- Statistics and analytics
- URL management
- Security features
- API integration
- QR Code generation
- URL preview
- Custom domains
- Expiry options
